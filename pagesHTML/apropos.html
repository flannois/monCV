Ce site à été réalisé en Python3 avec le framework Cherrypy. J'ai créé un petit modéle MVT pour me simplifié la tâche. J'ai choisi Python car je maitrise le language, et cherrypy car c'est un framework légé qui répondait parfaitement à ce projet. J'ai choisi de ne pas utiliser de base de données pour ne pas surcharger ce projet. Toutefois, je pense que cela pourra être une amélioration à apporter ultérieurement. J'ai également utilisé Bootstrap pour la mise en forme. Le serveur est héberger chez moi sur un Nas Synology, j'ai dû donc tout mettre en place pour que cela soit possible, paramètrage domaine, redirection de ports, gestion des firewalls, etc. Vous pourrez trouver le code source de ce site sur GitHub.
